# LOOPIT - Frontend

LOOPIT은 중고거래에서 발생할 수 있는 사기와 위협을 줄이기 위해 실시간 채팅, 신고 기능, 보안 강화 기능을 제공하는 보안 중심의 중고거래 플랫폼입니다. 이 저장소는 LOOPIT의 **프론트엔드(React)** 코드입니다.

---

## 📦 기술 스택

- ⚛️ React
- 🧭 React Router
- 🧾 Redux Toolkit
- 📦 Axios
- 💅 styled-components
- 💬 WebSocket (실시간 채팅)

---

## 🚀 주요 기능

- ✅ 상품 등록/조회/관리
- 💬 실시간 1:1 채팅 (판매자 - 구매자)
- 🚨 거래 중 이상 행위 신고 기능
- 🔍 거래 목록 및 채팅방 리스트 UI
- 🛡️ 사용자 인증 및 토큰 기반 보안

---

## 🛠️ 로컬 개발 환경 세팅

밑에 있는 입력어를 그대로 복붙해서 terminal에 넣어주세요.
```bash
git clone https://github.com/Jhkim6620/LOOPIT-front.git 
cd LOOPIT-front
npm install
npm start
```
